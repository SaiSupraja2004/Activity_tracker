# 🚀 Motivation Activity Tracker

A modern habit and activity tracking web application designed to help users stay consistent, track progress, and build productive habits.

## ✨ Features

- 🔥 Current Streak Tracking
- 🏆 Completion Rate Analytics
- 📈 Best Activity Detection
- 🎉 Confetti Celebration on Task Completion
- 💬 Feedback Notes for Every Activity
- 📅 Dynamic Date Management
- 🌙 Modern Dark UI
- 💾 Local Storage Persistence
- 📱 Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API
